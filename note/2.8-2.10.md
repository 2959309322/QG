# 2.8
### 关于`numpy`库

- 首先最基础的就是`numpy`中的基本数据类型**`ndarray`**

  它有很多在数据分析中很常用的**方法**，以下是他的一些

  **属性**

  |  属性   |      含义      |
  | :-----: | :------------: |
  | `shape` |      形状      |
  | `dtype` | 元素的数据类型 |
  | `ndim`  |   数据的维度   |

- 有一些**特性**

  |          |                             特性                             |
  | :------: | :----------------------------------------------------------: |
  |    -1    | 在获取数组ls[-1]的值时，这可以标识为<br>倒数第一行，倒数第一个...，即**负索引**。<br>在做reshape的一些操作时，可以指代生下所有元素 |
  |   切片   | 这个与python原生list类似<br>不同点在于`ndarray`还可以用`arr[x,x]`等价`arr[x][x]` |
  | 嵌套索引 | 可以用另一个`ndarray`作为索引参考列表`idx`放入<br>索引框内如`arr[idx]` |
  |   广播   | 一个简单的应用是神经网络线性层中偏置参与计算，<br>注意使用广播时，两个操作对象要至少有一个相同维度的相同参数。当然，对于零维的单独元素，它可以广播的条件很宽松 |

  *除此以外还有很多很多...*更详细的要去`numpy`官方文档中查看
